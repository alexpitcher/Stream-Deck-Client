# Stream Deck Client Projects

This repository houses the client software for the Stream Deck control plane system across multiple operating systems. The backend host runs on Windows (in a separate repository/module), while the clients connect over WebSockets to subscribe to physical stream deck UI changes and broadcast local system actions.

## Repository Layout

* [`/macos`](./macos) — The native Swift CLI client for macOS.
* `/win` — *(Coming in future phases)*
* `/linux` — *(Coming in future phases)*

See the respective directories for specific build and usage instructions.
